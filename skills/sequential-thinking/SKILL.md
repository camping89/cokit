# Sequential Thinking Skill

This skill teaches Copilot methodical problem solving.

## When to Activate

- Complex problem to solve
- Multi-step task
- User is stuck

## Approach

1. Break problem into smaller parts
2. Solve each part in order
3. Verify each step before next
4. Combine results
5. Verify final solution
