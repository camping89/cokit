# Planning Skill

This skill teaches Copilot to plan before implementing.

## When to Activate

- User asks to implement a feature
- Complex task requiring multiple steps
- User asks "how should I..."

## Approach

1. Understand requirements
2. Explore existing codebase
3. Design solution
4. Break into tasks
5. Identify risks
